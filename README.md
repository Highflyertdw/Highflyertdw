## Projects

### LoadCore — Work Order Management Platform
**Live:** [loadcore.vercel.app](https://loadcore.vercel.app)

A multi-tenant SaaS work order management system built for Coffee Creek Manufacturing (Oklahoma). Replaced a manual Google Sheets workflow with a full production application.

**What it does:**
- Work order creation and tracking across 13 dealer relationships
- Production status dashboard with real-time visibility into the shop floor
- Automated shop floor documentation and cover sheet generation
- Dealer profile management and order history
- Shipping schedule and calendar views
- Role-based access with multi-tenant data isolation

**Stack:** React 19, TypeScript, Supabase (PostgreSQL + Row Level Security), TanStack Query, React Router 7, Tailwind CSS, Vercel

**In development:** QuickBooks Online integration, VIN generation module
